## 🔍 Quick Reference

### Key Technologies

- **UI Framework:** Solid.js (30x faster than React)
- **2D Rendering:** PixiJS v8 (WebGL, automatic batching)
- **3D Rendering:** Three.js (first-person dungeons)
- **Animation:** GSAP
- **State Management:** TypeScript + Solid.js
- **Validation:** Zod
- **Build Tool:** Vite
- **Package Manager:** PNPM workspaces

### Project Stats

- **Documentation:** 8 files, ~260KB, 8,700+ lines
- **Game Content:** 2,000+ entries (spells, feats, items, monsters)
- **Performance Target:** 60 FPS @ 1920x1080
- **Rules System:** Full Pathfinder 2E Remastered
